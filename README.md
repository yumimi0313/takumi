# README
## 開発言語
- Ruby3.0.1
- Rails6.1.7

## 就業Termの技術
- devise
- rails_admin
- cancancan
- メッセージ機能
- フォロー機能

## カリキュラム外の技術
- chatGPT API
- ransack

## 実行手順
```
$git clone git@github.com:yumimi0313/takumi.git
$cd takumi
$bundle install
$yarn install 
$rails db:create && rails db:migrate
$rails s
```
